  4. array.concat()

    It is used combine/concatenate two or number of different arrays into one

    Example No. 4:

  5. array.copyWithin()

    It is used replace the items in the array. It is not used to add item but can only be used to replaced.

    Syntax:
    copyWithin(target, starting index, ending index)

    Example No. 5:
    const arrayCopy = ["amazing", "awsome", "fighting", "analayze", "attack", "advance"];
    console.log(arrayCopy.copyWithin(0,2,5));

    Output:

    ['fighting', 'analayze', 'attack', 'analayze', 'attack', 'advance']

    In above example, you can see that " .copyWithin() " replaced the items in the previous array " arrayCopy ".

  6. array.entries()

  19. array.join()

    It is used to join all the elements of an array and returns as a new string.

    Syntax:
    join(seperator)

    Example No. 19:
    const arrayJoin = ["animal", "plant", "dog"];
    console.log(arrayJoin.join());
    console.log(arrayJoin.join(":"));

    Output:

    animal,plant,dog animal:plant:dog

    In above example, you can see that " .join() " concatenates all the element and return a new string while on another one, we used seperator i.e ":" and the output is as expected.

  22. array.map()

    It returns the new array after calling the function for every element. It doesn't change the old elements.

    Example No. 22:
    const arrayMap = [1,2,3,4];
    console.log(arrayMap.map(x => x * 2));

    Output:

    [2,4,6,8]

    From the example, you can see that " .map() method returns new array after invoking a function for every elements.

  23. array.reduce()

    It is used to reduce the array to a single value and execute the function for each value of an array.

    Syntax:
    reduce(total, currentValue, index, array)

    Example No. 23:
    const arrayReduce = [2,4,8,16];
    console.log(arrayReduce.reduce((total,current) => total + current));

    Output:

    30

    From the example, you can see that " .reduce() method returns the total value after going through each elements of an array.

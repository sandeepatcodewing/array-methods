Different Array methods

  1. Array.form()

    It is used to break down all the characters or integer that is in single form. For eg: if there is a word "Hello" then it breaks into "H", "e", "l", "l", "o".

    Example No. 1:
    console.log(Array.form("amazing"));

    ['a','m','a','z','i','n','g']

  2. array.length

    It is used count the number of elements in array.

    Example No. 2:
    const arrayLength = ["amazing", "awsome", "fighting"];
    console.log(arrayLength.length);

    Output:

    3

    In above example, you can see that " .length " returns the number of elements inside an array.

  3. array.at()

    It is used to pinpot an exact item in the array

    Example No. 3:
    const arrayAt = ["amazing", "awsome", "fighting"];
    console.log(arrayAt.at(2));

    Output:

    fighting

    In above example, you can see that " .at() " pinpoint the exact item in the array and prints it.

  4. array.concat()

    It is used combine/concatenate two or number of different arrays into one

    Example No. 4:
    const arrayConcat1 = ["amazing", "awsome", "fighting"];
    const arrayConcat2 = ["analayze", "attack", "advance"];
    concatResult = arrayConcat1.concat(arrayConcat2);
    console.log(concatResult);

    Output:

    ["amazing", "awsome", "fighting", "analayze", "attack", "advance"]

    In above example, you can see that " .concate() " combines two array into one.

  5. array.copyWithin()

    It is used replace the items in the array. It is not used to add item but can only be used to replaced.

    Syntax:
    copyWithin(target, starting index, ending index)

    Example No. 5:
    const arrayCopy = ["amazing", "awsome", "fighting", "analayze", "attack", "advance"];
    console.log(arrayCopy.copyWithin(0,2,5));

    Output:

    ['fighting', 'analayze', 'attack', 'analayze', 'attack', 'advance']

    In above example, you can see that " .copyWithin() " replaced the items in the previous array " arrayCopy ".

  6. array.entries()

    It returns the key-value pairs of an index.

    Example No. 6:
    const arrayEntries = ["amazing", "awsome", "fighting"];
    for(const[index, element] of arrayEntries.entries()){
    console.log(index,element)
    }

    Output:

    0 'amazing' 1 'awsome' 2 'fighting'

    In above example, you can see that " .entries() " returns the key-value pair of an index " 0 , 1 , 2 ".

  7. array.every()

    It returns boolean value i.e true or false. It returns true if it satisfies the function.

    Example No. 7:
    const arrayEvery = [31,21,34,22,29,37];
    console.log(arrayEvery.every(greaterThan));
    function greaterThan(num){
    return num > 20;
    }

    Output:

    true

    In above example, you can see that " .every() " statisfied the function and returns " true ".

  8. array.fill()

    It is used to replace the items in the array. It is similar to " copyWithin() " method but instead of trageting the index of an array to change elements it provides the value and starting index of an array and end index of an array.

    Syntax:
    fill(value, starting index, ending index)

    Example No. 8:
    const arrayFill = [31,21,34,22,29,37];
    console.log(arrayFill.fill(0,1,3));

    Output:

    [31, 0, 0, 22, 29, 37]

    In above example, you can see that " .fill() " replaced the items in the previous array " arrayFill ".

  9. array.filter()

    It is used to filter the elements as it required.

    Example No. 9:
    const arrayFilter = [31,21,34,22,29,37];
    console.log(arrayFilter.filter(moreThan));
    function moreThan(num){
    return num > 26;
    }

    Output:

    [31, 34, 29, 37]

    In above example, you can see that " .filter() " filtered and displays the desired output.

  10. array.find()

    It is used to find the element in an array if certain requirements meets or the functions get statisfied.

    Example No. 10:
    const arrayFind = [6,8,16,31,21,34,22,29,37];
    console.log(arrayFind.find(maxNum));
    function maxNum(num){
    return num > 20;
    }

    Output:

    31

    In above example, you can see that " .find() " finds the number which is greater than 20 according to the function "maxNum()". It is clearly seen that it scans the number from left to right and prints the first number or value when it matches with the function requirements
